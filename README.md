# Stock-Market-Intelligence
Problem Statement:Investors often struggle to interpret raw stock prices.
This system converts raw market data into meaningful indicators and insights, helping users understand trends, momentum, risk, and performance more easily.

~The project demonstrates:
API integration (yfinance)
Data analysis with Pandas
Financial analytics
Data visualization
Dashboard development with Streamlit
Real-time data processing
Problem-solving and analytical thinking

~How it works:
Fetches live stock data from Yahoo Finance.
Processes the data using Pandas.
Performs technical analysis:
Moving Averages
RSI
Volatility
Returns Analysis

Displays interactive charts and metrics using Streamlit.

Generates automated insights to simplify decision-making.
