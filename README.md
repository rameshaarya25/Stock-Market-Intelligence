Problem Statement:
Investors often struggle to interpret raw stock prices. This system converts raw market data into meaningful indicators and insights, helping users understand trends, momentum, risk, and performance more easily.

This project is a real-time stock market analytics dashboard that fetches one year of live stock data from Yahoo Finance and performs technical analysis.

How it works:
Fetches live stock data from Yahoo Finance.
Processes the data using Pandas.
Performs technical analysis:
Moving Averages
RSI
Volatility
Returns Analysis
Displays interactive charts and metrics using Streamlit.
Generates automated insights to simplify decision-making.
