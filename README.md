#Stock-Market-Intelligence

A real-time stock market analytics dashboard built using Python, Streamlit and Yahoo Finance API.

The platform performs:

- Live Stock Tracking
- RSI Analysis
- Moving Average Analysis
- Volatility Analysis
- Automated Insights

Features:

- Live Market Data
- Market Status
- Daily Return Analysis
- RSI Indicator
- 50-Day Moving Average
- 200-Day Moving Average
- Volatility Analysis
- Automated Insights

 Technologies Used:

- Python
- Streamlit
- Pandas
- Yahoo Finance API
